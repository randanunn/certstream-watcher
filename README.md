# Carrie CertStream

### System Requirements
* Java 17
* Maven

#### Run Maven Package
```
mvn package
```

#### Build Docker Image
```
docker build -t name_of_image .
```

#### Run Docker Image Locally
```
docker run --rm name_of_image
```

#### Notes
* Install Azure CLI for pushing image to Azure
* See Azure Container docs for how to tag and push image
